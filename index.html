🚀 GitHub Pages 배포 가이드 — 처음 설정하는 방법 (클릭해서 펼치기)
1
GitHub 계정 만들기 / 로그인
github.com에서 무료 계정을 만드세요. 친구들도 계정이 있으면 댓글 기능 이용 가능합니다.
2
New Repository 생성
우측 상단 "+" → "New repository" 클릭. 이름을 trading-dashboard로 설정하고 Public으로 만드세요.
3
이 HTML 파일 업로드
Repository 메인 페이지에서 "Add file" → "Upload files" 클릭. 이 HTML 파일을 index.html로 이름 바꿔서 업로드하세요.
4
GitHub Pages 활성화
Repository → Settings → 왼쪽 메뉴 Pages → Source: Deploy from a branch → Branch: main → Save
5
링크 공유!
1~2분 후 아래 주소로 접근 가능합니다. 친구들에게 이 링크를 공유하세요!
https://YOUR_GITHUB_USERNAME.github.io/trading-dashboard/
6
뉴스 업데이트 방법
GitHub에서 직접 index.html 파일의 NEWS_DATA 배열을 수정하면 즉시 업데이트됩니다. 주가는 자동으로 실시간 로드됩니다.
💡 팁: 친구들이 "Trading Opinions" 섹션에서 의견을 남기면 각자 브라우저에 저장됩니다. 서로 같은 화면을 보면서 화상통화나 카톡으로 의견을 나누는 방식도 좋습니다!
